# 问题排查记录

## 问题 1: 倒计时不显示
**现象**: 准备后没有倒计时显示

**原因**: `public/index.html` 第 137 行拼写错误
```javascript
countdownDiv.textContent = `倒计时: ${data.countdow}秒`;
```

**解决**: 改为 `data.countdown`

---

## 问题 2: 所有玩家都显示拳头，无法计分
**现象**: 无论出什么，结果都显示拳头，没有得分

**原因**: `public/index.html` 第 164 行未使用 `await`
```javascript
btn.addEventListener('click', () => {
  const hash = sha256(myChoice + mySalt); // 返回 Promise 对象
});
```

**解决**: 添加 `async/await`
```javascript
btn.addEventListener('click', async () => {
  const hash = await sha256(myChoice + mySalt);
});
```

---

## 问题 3: node_modules 被提交到 git
**现象**: 依赖文件夹被推送到 GitHub

**解决**:
```bash
# 创建 .gitignore
echo "node_modules/" > .gitignore

# 从 git 移除
git rm -r --cached node_modules
git commit -m "Remove node_modules"
```

---

## 问题 4: OAuth 错误 - invalid_client
**现象**: Google 登录显示 "错误 401: invalid_client"

**原因**: `.env` 文件中使用占位符，未配置真实凭据

**解决**:
1. 在 Google Cloud Console 创建 OAuth 客户端
2. 更新 `.env` 文件中的真实 Client ID 和 Secret
3. 或者不配置 OAuth，使用昵称登录

---

## 问题 5: Railway 显示 Internal Server Error
**现象**: Railway 部署后访问报错

**原因**: Railway 环境变量未配置

**解决**: 在 Railway Variables 添加：
- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`
- `CALLBACK_URL`
- `SESSION_SECRET`

或者代码改为 OAuth 可选（已修复）
