# 石头剪刀布游戏 - 部署说明

## 项目地址
- GitHub: https://github.com/Alanzzh/rock-paper-scissors
- Railway: https://rock-paper-scissors-production-0b2f.up.railway.app

## 本地运行
```bash
cd C:/Users/admin/rock-paper-scissors
npm install
npm start
```
访问: http://localhost:3002

## Railway 部署
1. Railway 自动从 GitHub 部署
2. 每次推送到 main 分支会自动重新部署
3. 部署时间约 1-2 分钟

## Google OAuth 配置

### Railway 环境变量
在 Railway 项目的 Variables 标签添加：
```
GOOGLE_CLIENT_ID=你的Client ID
GOOGLE_CLIENT_SECRET=你的Client Secret
CALLBACK_URL=https://rock-paper-scissors-production-0b2f.up.railway.app/auth/google/callback
SESSION_SECRET=rock-paper-scissors-secret-key-2024
```

### Google Cloud Console 设置
1. 访问 https://console.cloud.google.com/
2. 创建项目
3. 启用 Google+ API
4. 创建 OAuth 2.0 客户端 ID
5. 添加授权重定向 URI:
   - 本地: `http://localhost:3002/auth/google/callback`
   - Railway: `https://rock-paper-scissors-production-0b2f.up.railway.app/auth/google/callback`

详细步骤见 `GOOGLE_OAUTH_SETUP.md`

## 注意事项
- 不配置 Google OAuth 也能正常使用（通过昵称登录）
- 推送代码前确保网络连接正常
- Railway 会自动检测 GitHub 更新并部署
